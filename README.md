## Chapter 2 변수
println()과 print()의 차이점
- println(): 줄바꿈 O
- print(): 줄바꿈 X
기본형과 참조형 변수
- 기본형 변수: 실제 값 저장(총 8개 타입)
- 참조형 변수: 메모리 주소를 저장(포인터)
참조형 변수
- `Date today = new Date();`
- Date는 클래스 이름(자료형), today는 변수명
- new는 객체를 생성하는 연산자, 그 결과는 생성된 객체의 주소
상수
- 값 변경 불가능
- `final int SEONGMIN_AGE = 25;`
